\# HardwareProject



This project sorts Sri Lankan coins and notes using sensors and a microcontroller.



\## Features

\- Detects coins: 1, 2, 5, 10 LKR

\- Detects notes: 20, 50, 100, 1000, 5000 LKR

\- Shows real-time totals on a web dashboard



\## Technologies

\- Arduino Uno

\- IR Sensors

\- Load Cell

\- PHP, MySQL
